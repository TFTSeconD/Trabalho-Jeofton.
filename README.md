# Sistema de Controle de Estoque



Projeto prático desenvolvido para a disciplina de Programação Web.

O sistema realiza o controle de estoque de produtos simples (CRUD).



## Tecnologias Utilizadas

- PHP (Sem frameworks)

- Javascript (Puro)

- Bootstrap 5

- MySQL



## Como rodar

1. Importe o script SQL no seu banco de dados.

2. Configure o arquivo `db.php`.

3. Acesse via servidor Apache.
